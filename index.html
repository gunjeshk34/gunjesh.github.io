About Me

I am skilled in crafting visually appealing and highly functional websites that enhance user experience and engagement. My expertise spans across a wide range of essential tools and technologies, including:

  • HTML & CSS: I have a deep understanding of HTML and CSS, allowing me to create clean, semantic, and well-structured code that forms the backbone of responsive and accessible web pages.
  • JavaScript & jQuery: My proficiency in JavaScript and jQuery enables me to add dynamic, interactive elements to websites, enhancing usability and interactivity with smooth animations and effective user interactions.
  • Bootstrap: Using the Bootstrap framework, I can efficiently build mobile-first, responsive websites that adapt perfectly to different devices and screen sizes, providing a consistent user experience.
  • Photoshop: My skills in Adobe Photoshop allow me to create and edit high-quality graphics and visual elements, ensuring that each design is polished and visually appealing.
  • Figma: With Figma, I can create wireframes and prototypes, collaborate in real-time, and design intuitive interfaces that align with client needs and project goals.
